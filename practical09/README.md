# Magic Square

This programme tests arrays contained in text files to check if they contain a magic square

## To compile

gcc main.c -o main


## Two examples of running this 

### Example 1

./main 

Enter file name: magic_square.txt

No. lines, 3

This is magic 

### Example 2

./main 

Enter file name: not_magic_square.txt

No. lines, 3

This is not magic
